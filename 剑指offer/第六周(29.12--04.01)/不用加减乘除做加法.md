# 剑指offer 65. 不用加减乘除做加法



### 题目地址：[不用加减乘除做加法](https://leetcode-cn.com/problems/bu-yong-jia-jian-cheng-chu-zuo-jia-fa-lcof/)



### 题目描述：

写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。



### 解答方法：

1. 

公式 ： 没有进位的加法：（a ^ b)

​					进位：(a & b) << 1;

​                直到进位为 0，则“（a ^ b)”为最终结果

```java
class Solution {
    public int add(int a, int b) {
        int c = a^b;
        int d = (a&b) << 1;
        if(b == 0){
            return a;
        }
        return add(c^d,(d&c)<<1);
    }
}
```
